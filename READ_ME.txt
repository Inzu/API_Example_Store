FOREWORD

The purpose of this code is to allow you to quickly add store pages to your site. The pages can be re-designed however you like.


INSTALLATION

1. Upload all files, excluding this 'Read Me' file to the root directory of your website or sub-domain.

2. Change the config settings to your settings or use the sandbox account for testing.

3. Load the index.php file in your browser.

You should now see content from the Inzu testing environment.

Change the code in these files to suit your site design.


FILES

cart.js - this script manages the cart cookie for adding items and editing the cart.
cart_edit.php - let users edit their cart.
config.php - your API account settings and some shop settings such as currency and callback URL.
index.php - display an example store page.
style.css - styles for this example.

For more information on the Inzu API and to reference every API feed please visit http://developers.inzu.net or contact support@inzu.net.
